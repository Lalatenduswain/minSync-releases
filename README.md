# minSync — Releases

Download the latest version of **minSync**, the cross-platform mobile client for self-hosted MinIO storage.

## What is minSync?

minSync connects your phone directly to your self-hosted MinIO (S3-compatible) server, providing:

- **True background sync** via Android WorkManager & iOS BGTaskScheduler
- **In-app file streaming** — view images, play videos, read documents without full downloads
- **Secure credential storage** — iOS Keychain & Android EncryptedSharedPreferences
- **Optional E2E encryption** — AES-256-GCM, encrypted before leaving your device
- **Delta sync** — only sync what changed, minimal bandwidth

## Installation (Android)

1. Download the latest `.apk` from [Releases](https://github.com/Lalatenduswain/minSync-releases/releases)
2. Enable "Install from unknown sources" in your device settings
3. Open the downloaded APK and install
4. Launch minSync and add your MinIO server
